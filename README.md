# BOOK-ECOMMERCE-UI-UX-DESIGN


This project showcases a mobile-based eCommerce website UI/UX design created in Figma, focusing on a smooth and intuitive shopping experience, with user onboarding, account and profile management, easy navigation menus, detailed product views, location and delivery selection, order confirmation flow, attractive offers display, and a responsive help and support section across 10 well-structured pages.
